Restaurant Management App  
A Kotlin + Jetpack Compose mobile application for managing restaurant operations.

Overview  
The Restaurant Management App is a demo mobile application built entirely with **Kotlin** and **100% Jetpack Compose**. It provides a simple but functional structure for managing different roles within a restaurant environment, including:

- ✅ **Admin Management**  
- ✅ **Employee Management**  
- ✅ **Customer Management**

This project was developed using **Android Studio**, following modern Android development practices. It serves as a foundation for building a full restaurant operations system.

> ⚠️ *Note:* Some Gradle files and dependencies are slightly outdated, so updating them before extending the project is recommended.


🚀 Features

👨‍🍳 Admin Module
- Manage employees  
- Manage menu items  
- View restaurant activity  

👥 Employee Module
- Handle customer orders  
- Update order status  
- View assigned tasks  

🧑‍💼 Customer Module
- Browse menu  
- Place orders  
- View order history  

🛠️ Technical Features
- Built with **Kotlin**  
- UI built using **Jetpack Compose**  
- Modular structure for Admin, Employee, and Customer flows  
- Uses Android Studio Gradle build system

▶️ Getting Started

### ✅ Prerequisites
- Android Studio Flamingo or newer  
- JDK 17+  
- Android SDK 33+  

### ✅ Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Nuel-09/Restaurant_Management_App.git
   ```
2. Open the project in **Android Studio**  
3. Let Gradle sync  
4. Run the app on an emulator or physical device
